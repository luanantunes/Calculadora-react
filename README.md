🧮 Calculadora React

Uma calculadora moderna desenvolvida com React, focada em design limpo e experiência do usuário.

![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-f7df1e?style=flat-square&logo=javascript)

---

## ✨ Funcionalidades

- Operações básicas: soma, subtração, multiplicação e divisão
- Histórico dos últimos cálculos (clicável)
- Suporte completo ao teclado
- Design responsivo com tema dark
- Animações e feedback visual nos botões

---

## 🛠️ Tecnologias utilizadas

- **React 18** — biblioteca principal
- **React Hooks** — useState, useEffect, useCallback
- **CSS-in-JS** — estilização inline com variáveis
- **JavaScript ES6+** — lógica de cálculo

---

## 🚀 Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/luanlima/calculadora-react

# Entre na pasta
cd calculadora-react

# Instale as dependências
npm install

# Rode o projeto
npm start
```

Acesse em: `http://localhost:3000`

---

## 💡 O que aprendi com esse projeto

- Gerenciamento de estado com múltiplos `useState`
- Como usar `useCallback` para otimizar funções
- Captura de eventos de teclado com `useEffect`
- Organização de componentes e lógica de negócio no React
